About the Course Project

This is a deliverable for the course Developing Data Products as part of the Coursera Data Science Specialization.

Instructions:

  1. Write a shiny application with associated supporting documentation. The documentation should be thought of as whatever a user will need to get started using your application.
  2. Deploy the application on Rstudio’s shiny server
  3. Share the application link by pasting it into the provided text box
  4. Share your server.R and ui.R code on github

How to use the application

Using the data provided by Galton Families dataset, we fit a linear model to predict a child’s height based on the gender and parent’s average height.

The application is running on (https://rb.gy/v3utyw)

ui.R, and Server.R Code in my github repository (https://github.com/bulahuang/Course-Project-Shiny-Application-and-Reproducible-Pitch)
